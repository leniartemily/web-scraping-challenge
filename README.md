## General info
This challenge was to scrape the [Mars News Site](https://redplanetscience.com/) and collect the latest News Title and Paragraph Text. 
	
## Technologies
Project is created with:
* Jupyter Notebook
* BeautifulSoup
* Requests/Splinter.
